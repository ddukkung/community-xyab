# XYAB
![xyab](https://user-images.githubusercontent.com/88926356/162677238-11d72030-e401-433d-b872-824b849d0e54.jpg)

* XYAB는 닌텐도 스위치 유저를 위한 게임 커뮤니티입니다.
* 회원가입이 가능하며 자신의 글을 작성하여 다른 사람들과 공유할 수 있습니다.

***

# 개발환경

  * Java 11
  * Spring Boot 2.6.4
  * IntelliJ
  * MySQL 5.7
  * JPA
  * Spring Security
  * Thymeleaf
  * Ajax, BootStrap
